## fetch_telemetry.py

Pre-conditions:

- export Kubeconfig
- change context
- change namespace

Run like this:

```bash
python cli.py <CLUSTER_ID> --output-dir ./out
```

Alternatively, if you have already downloaded the files, you can run the Node Report separately:

```bash
python node_report.py
```
